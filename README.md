## Run

### Regular Run

```bash
hugo server
``` 

*Note: It generally serves in port 1313 -> http://localhost:1313*


### Run in local IP

```bash
hugo server --bind <localIP> --baseURL http://<localIP>:<port>

hugo server --bind 192.168.1.1 --baseURL http://192.168.1.1:1313
```
